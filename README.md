# Docentra PDF Editor

🎯 **Docentra PDF Editor** - Güçlü ve Kullanıcı Dostu PDF Düzenleyici Uygulaması

[![Website](https://img.shields.io/badge/Website-docentrapdf.com-blue?style=flat-square)](https://docentrapdf.com)
[![License](https://img.shields.io/badge/License-Proprietary-red?style=flat-square)](#lisans)

## 📋 Hakkında

Docentra PDF Editor, web tabanlı modern bir PDF düzenleyici uygulamasıdır. Kullanıcılar doğrudan tarayıcıdan PDF dosyalarını görüntüleyebilir, düzenleyebilir ve işlemci yapabilirler.

### ✨ Özellikler

- 📄 PDF görüntüleme ve düzenleme
- 🖊️ Ek anotasyon araçları
- 💾 Geliştirilmiş dosya yönetimi
- ⚡ Hızlı ve responsive tasarım
- 🔒 Güvenli işlem

## 🚀 Kullanım

Web sitesini ziyaret edin: **[docentrapdf.com](https://docentrapdf.com)**

Uygulamayı kullanmak için herhangi bir kurulum gerekmez. Sadece tarayıcınızda açın ve başlayın!

```
DocentraWebsite/
```

## 🛠️ Teknoloji Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Hosting:** GitHub Pages
- **Domain:** Custom domain (docentrapdf.com)
